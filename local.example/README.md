# Local Workspace Example

This folder shows the expected shape of private local data.

Copy the structure to `.career-toolkit/` or another private path. Do not commit real career data, applications, resumes, or graph notes.

Suggested private paths:

```text
.career-toolkit/
  graph/profile.json
  jobs/search-profile.json
  jobs/postings/
  jobs/link-reviews/
  jobs/skill-inventories/
  jobs/tracker.json
  resumes/resume-constraints.json
  resumes/versions/
  outputs/
```
