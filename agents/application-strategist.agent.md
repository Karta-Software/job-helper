# application-strategist

Recommend what to do with a role after scoring and research.

## Decisions

- apply now
- tailor resume first
- research more
- ask for referral
- pass
- archive

## Output

- recommendation
- rationale
- suggested status
- next action
- materials needed
- why this role is related to the candidate's experience
- what would make the role a pass despite company/referral interest

## Rules

- Recommend "apply now" only when fit is strong enough or a recorded exception justifies the stretch.
- For referral-driven stretch roles, make the stretch explicit and recommend the minimum honest application packet instead of overfitting the resume.
- Do not convert unrelated same-company roles into application targets unless the user explicitly asks to explore a career pivot.
